# 한글 색칠 놀이터

좋아하는 캐릭터 도안 옆에 한글 글자를 외곽선으로 칸칸이 배치한 A4 가로 학습지를 만드는 단일 페이지 도구.

## 사용법

1. **단어 입력** — 헤더의 "단어" 칸에 따라 쓸 한글 단어(예: `하츄핑`, `토르`)를 적는다. 오른쪽 그리드가 글자 수에 맞춰 자동으로 N열 × 3행으로 다시 그려진다.
2. **이름 / 날짜** — 학습지 좌측 상단에 표시된다. 비워두면 점선 자리만 보인다.
3. **도안 넣기** — 다음 세 가지 방법:
   - `그림` 버튼으로 파일 업로드
   - 다른 곳에서 이미지를 복사하고 페이지에서 `Ctrl + V`
   - 도안 슬롯 위로 이미지 파일을 드래그&드롭
4. **저장**
   - `PDF` — A4 가로 한 페이지로 다운로드
   - `프린트` — 브라우저 인쇄 (이미 가로 A4로 설정됨)

## 기술

- 단일 HTML, vanilla CSS/JS
- CDN ESM: `html2canvas@1.4.1`, `jspdf@2.5.1` (esm.sh)
- 폰트: Pretendard (CDN)
- React 프로토타입: [`sample_prototype/`](sample_prototype/) — 참고용으로 함께 두었으며 배포본은 아님.

## iOS 홈 화면

Safari → 공유 → 홈 화면에 추가 → 풀스크린 앱처럼 열린다.
(이미 추가한 적이 있으면 한 번 지우고 다시 추가해야 풀스크린이 적용된다.)
