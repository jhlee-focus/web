## 2. 핵심 콘셉트

> 캐릭터 친구들이 디저트 가게를 열었어요.
> 아이들은 친구들을 도와 맛있는 디저트를 만들고, 예쁘게 꾸며 완성합니다.

이 게임은 이기고 지는 게임이 아니라, **아이들이 직접 고르고 만드는 놀이형 게임**이다. 실패보다는 **칭찬, 완성, 꾸미기, 반복 플레이**에 집중한다.

### 설계 3원칙
1. **실패하지 않는다** — 잘못 눌러도 끝나지 않고 부드럽게 안내
2. **글을 못 읽어도 진행된다** — 안내 손가락과 깜빡임으로 다음 동작 표시
3. **꾸미기에서 가장 오래 머문다** — 자유 배치 모드로 정답 없는 시간 제공

## 4. 기본 플레이 흐름

```text
시작 화면
  ↓
캐릭터 등장 + 인사
  ↓
디저트 선택 (2개)
  ↓
재료 고르기 (3-5개 탭)
  ↓
도구 사용하기 (탭 N회 또는 탭 후 자동)
  ↓
오븐/냉장고 (탭 후 자동 진행)
  ↓

  ↓
캐릭터 칭찬 + 별 3개
  ↓
[다시 만들기 / 다른 디저트 / 처음으로]
```

## 5. 조작 동사 정의 (v0.3 신설)

비개발자가 AI 코딩 도구에 명확히 지시할 수 있도록, **모든 조작은 다음 4가지로만 통일**한다. 드래그는 v0.1에서 사용하지 않는다 (6세 아이는 손가락 떼는 타이밍이 어려움).

| 조작 동사 | 정의 | 사용 장면 |
|---|---|---|
| **단일 탭** | 한 번 누르기 | 버튼, 재료 선택, 도구 선택 |
| **반복 탭** | N회 누르기 (보통 3-5회). 누를 때마다 시각 피드백(흔들림/색 변화) + 진행 게이지 채워짐 | 반죽 섞기, 크림 짜기 |
| **탭 후 자동** | 한 번 탭하면 애니메이션이 자동 진행 (1.5-3초) | 오븐 굽기, 냉장고 식히기, 재료 그릇에 들어가기 |
| **자유 탭 배치** | 토핑 팔레트에서 토핑을 탭 → 캔버스 위 원하는 위치를 탭하면 그 자리에 토핑 배치 | 꾸미기 단계 |

### 안내 손가락 표준 동작
- 다음에 눌러야 할 위치에 **반투명 손가락 아이콘 + 깜빡임**
- **3초간 아무 입력 없으면** 자동으로 깜빡임 시작
- 정답이 아닌 곳을 누르면 안내 손가락이 정답 위치로 이동하며 강조

## 6. 화면 구성

### 6-1. 시작 화면
| 요소 | 설명 |
|---|---|
| 게임 제목 | "티니핑 디저트 가게" 큰 글씨 |
| 시작 버튼 | 화면 하단 중앙, 큰 둥근 버튼 |
| 배경 | 파스텔톤 디저트 가게 |
| 캐릭터 | 화면 좌측에서 손 흔드는 안내 캐릭터 |
| 안내 손가락 | 시작 버튼 위 깜빡임 |

### 6-2. 디저트 선택 화면
2장의 디저트 카드를 세로로 큼직하게 배치 (세로 모드 기준).

```text
[ 초코칩 쿠키 ]
[   케이크   ]
```

| 요소 | 설명 |
|---|---|
| 디저트 그림 | 완성 디저트 이미지 |
| 디저트 이름 | 큰 한글 |
| 선택 효과 | 탭하면 카드가 살짝 커지며 반짝 → 다음 화면 |
| 홈 버튼 | 우상단 작은 아이콘 (확인 팝업 후 시작 화면으로) |

### 6-3. 재료 선택 화면
| 요소 | 설명 |
|---|---|
| 재료 팔레트 | 화면 하단에 큰 아이콘 4-6개 |
| 그릇 | 화면 중앙 |
| 안내 캐릭터 | 좌측 상단 말풍선으로 "밀가루를 골라줘!" 등 한 가지씩 안내 |
| 진행 표시 | 상단에 "1/4, 2/4..." 같은 점 표시 |

**잘못된 재료 탭 시**: 흔들림 효과 + 말풍선 "이건 다음에 써보자!" → 게임은 진행됨, 다시 탭 가능.

### 6-4. 만들기(조리) 화면
재료가 그릇에 다 들어가면 자동 전환.

| 단계 | 조작 |
|---|---|
| 섞기 | 그릇/스푼을 5회 반복 탭 → 진행 게이지 채워지면 자동 다음 |
| 굽기 | 오븐 탭 → 1.5초 자동 진행 + 연기 효과 |
| 식히기 | (해당 디저트만) 냉장고 탭 → 1.5초 자동 |

### 6-5. 꾸미기 화면 (v0.3 신설)
**가장 자유롭고 가장 오래 머무는 단계**.

| 요소 | 설명 |
|---|---|
| 캔버스 | 화면 상단 2/3, 완성 직전의 디저트 |
| 토핑 팔레트 | 화면 하단, 토핑 5-8종 (무한 사용) |
| 배치 방법 | 토핑 탭 → 디저트 위 원하는 위치 탭 → 그 자리에 배치 |
| 되돌리기 | 우하단 "↩" 버튼, 마지막 토핑 제거 |
| 완성 버튼 | 우하단 큰 "완성!" 버튼 (언제든 누를 수 있음, 토핑 0개여도 OK) |
| 정답 없음 | 어떻게 꾸며도 칭찬 |

### 6-6. 완성 화면
| 요소 | 설명 |
|---|---|
| 완성 디저트 | 꾸민 그대로 크게 표시 |
| 별 3개 | 무조건 3개, 순차적으로 반짝이며 등장 |
| 캐릭터 칭찬 | "와! 정말 맛있겠다!" / "예쁘게 만들었네!" 등 (랜덤) |
| 반짝이 효과 | 화면 전체에 별/하트 파티클 |
| 버튼 3개 | [다시 만들기] [다른 디저트] [처음으로] |

## 9. 화면/스타일 가이드

### 9-1. 화면 비율
- **세로 모드 고정** (폰 세로로 들고 플레이)

### 9-2. 색상 팔레트

### 9-4. 버튼 규격

## 13. 구현 방식 (v0.1)

### 13-1. 권장 방식
**HTML/CSS/JavaScript 단일 파일** (index.html 하나로 끝)

## 14. AI 코딩 도구용 요청 초안 (v0.3 갱신)

```text
6세 아이를 위한 디저트 만들기 웹 게임을 단일 HTML 파일로 만들어줘.

[게임 컨셉]

- 대상: 만 3-6세
- 폰/태블릿 브라우저 세로 모드 고정
- 탭 조작만 사용 (드래그 금지)
- 실패 없는 구조 — 잘못 눌러도 게임 진행됨, 부드럽게 안내

[v0.1 디저트 2개]
1. 초코칩 쿠키 (단순 흐름 검증)
2. 케이크 (꾸미기 자유도 검증)

[조작 동사 4가지]
- 단일 탭: 버튼, 재료 선택
- 반복 탭: 5회 누르면 진행 게이지 가득 차며 자동 다음
- 탭 후 자동: 오븐/냉장고 등, 한 번 탭 후 1.5-3초 자동 애니메이션
- 자유 탭 배치: 꾸미기 단계, 토핑 탭 → 위치 탭 → 그 자리에 배치

[화면 흐름]
시작 → 디저트 선택(2개) → 재료 고르기(4개 순차) → 섞기(반복 탭) → 
굽기(자동) → (케이크만 크림 바르기) → 꾸미기(자유 배치) → 
완성(별 3개 고정 + 칭찬 + 반짝이) → [다시/다른/처음] 버튼

[필수 UI]
- 모든 화면 우상단 홈 아이콘 (확인 팝업 후 시작 화면)
- 안내 손가락 (다음 액션 위치에 깜빡임, 3초 무입력 시 자동 시작)
- 진행 점 표시 (재료 선택 시 1/4, 2/4...)
- 캐릭터 말풍선 (좌측 상단)

[비주얼]
- 파스텔톤 (배경 #FFF5F7, 주요 버튼 #FF8FAB, 강조 #FFD93D)
- 둥근 큰 버튼 (최소 60x60px, border-radius 16-24px)
- 텍스트 18px+ , 버튼 24px+
- 캐릭터/재료/도구는 일단 이모지 placeholder로 구현
- CSS 애니메이션으로 반짝이/흔들림/하트 파티클 구현

[제외 사항 (v0.2 이후)]

- 별점 차등 없음 (무조건 3개)
- 저장/도감 없음
- 드래그 조작 없음

[코드 구조]
- DESSERTS, INGREDIENTS, PRAISES 객체로 데이터 분리
- 캐릭터는 character_guide 변수로 추상화 (나중에 교체 용이)
```

## 15. 테스트 체크리스트 (v0.3 신설)

만들고 나서 아이와 함께 플레이할 때 관찰할 항목.

### 15-1. 자력 진행 가능 여부
- [ ] 시작 버튼을 알려주지 않고도 누르나
- [ ] 디저트 카드를 자기가 골라서 누르나
- [ ] 재료 선택 시 "다음 뭐 누르지?"를 안내 손가락 보고 알아채나
- [ ] 반복 탭 단계에서 충분히 누르나 (지루해하지 않나)
- [ ] 꾸미기 단계에서 토핑을 자유롭게 배치하나

### 15-2. 감정 반응
- [ ] 완성 화면에서 웃거나 박수 치나
- [ ] 별이 등장할 때 좋아하나
- [ ] 잘못된 재료를 눌렀을 때 좌절하나, 자연스럽게 다른 재료를 누르나
- [ ] 끝난 뒤 "또 할래"라고 말하나

### 15-3. 멈칫 지점
- [ ] 어느 화면에서 가장 오래 멈춰 있나
- [ ] 어느 단계에서 부모에게 도움을 청하나
- [ ] 어떤 토핑을 가장 좋아하나 (v0.2 토핑 추가 시 참고)

### 15-4. 부모 관찰
- [ ] 부모가 "이제 그만"이라고 했을 때 홈 버튼이 잘 작동하나
- [ ] 같은 디저트를 몇 번 반복하나
- [ ] 두 디저트 중 어느 쪽을 더 자주 고르나

## 최종 요약

이 게임은 **캐릭터와 함께 초코칩 쿠키와 케이크를 만드는 어린이용 디저트 만들기 게임**(v0.1)이다.

핵심은:

> **고르기 → 넣기 → 섞기(반복 탭) → 굽기(자동) → 꾸미기(자유 배치) → 완성 → 칭찬**

이 흐름을 아이가 **글을 못 읽어도, 실수해도, 혼자서** 즐길 수 있게 만드는 것이다.

v0.1로 패턴을 검증하고, 아이가 좋아한 부분을 강화하며 v0.2에서 디저트 4개와 사운드를 추가한다.
